選擇執行模式

`Local` 適合本機 exe，會用 EventSub WebSocket。`Server` 適合可公開存取的伺服器，會用 EventSub Webhook。

Twitch App 建議

  1. Local Mode: 建議註冊為公開 client，使用 Device Code Flow。
  2. Server Mode: 建議註冊為機密 client，使用 Authorization Code Flow。
  3. Redirect URI 只有 Server Mode 的 OAuth popup 會用到，例如 http://localhost:3000/setup/callback。
